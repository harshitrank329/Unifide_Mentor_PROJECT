# Nassau Candy Factory Reallocation & Shipping Optimization

This is a student-friendly data science project built from the Nassau Candy Distributor CSV.
It shows how to use a real business dataset to:

- clean and prepare order data
- predict shipping lead time
- compare alternate factory assignments
- rank simple reassignment recommendations
- explain results in a Streamlit dashboard

## What students learn

- feature engineering with dates, categories, and numeric values
- train/test splitting for time-based business data
- comparing baseline and tree-based regression models
- scenario analysis and simple optimization logic
- dashboard design for business communication

## Quick start

1. Install the dependencies from `requirements.txt`.
2. Run the dashboard:

```bash
streamlit run app.py
```

## Project files

- `app.py`: the Streamlit dashboard
- `nassau_optimization/`: reusable analysis code
- `Nassau Candy Distributor.csv`: source dataset

Use this project as a case study to show how predictive analytics can support factory reassignment decisions,
